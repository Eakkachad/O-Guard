# SkillBridge: Bridging the Gap Between Talent and Industry

> An intelligent, data-driven learning path and job placement platform addressing structural unemployment through AI-powered skill gap analysis.

## 📖 Overview

**SkillBridge** (สะพานทักษะ เชื่อมคนสู่งาน) is a conceptual and technical framework designed to bridge the growing disconnect between industrial labor demands and the current workforce skill outturn in Thailand.

By operating as a two-sided marketplace, the platform intelligently connects **Opportunity Seekers** (individuals requiring upskilling/reskilling) with **Employers** (factories, enterprises) by analyzing market demands, defining concrete skill gaps, and generating specialized, predominantly free learning paths.

### 🎯 Core Objectives

- **Mitigate Structural Unemployment:** Directly address the mismatch between available jobs and current workforce capabilities.
- **Promote Lifelong Learning:** Provide accessible, low-cost/free educational routes aggregating resources from MOOCs and Thai Government frameworks (e.g., Department of Skill Development).
- **Guaranteed Placement:** Incentivize learning through a system that guarantees or highly prioritizes job placement upon successful path completion.

---

## 🏗️ System Architecture

The SkillBridge ecosystem comprises several interconnected, AI-driven modules designed to guide a user from initial assessment to final employment:

1. **AI Matching Engine:** Utilizes Natural Language Processing (NLP) to parse corporate job descriptions and extract standardized required skills.
2. **Skill Gap Analyzer:** Employs classification algorithms to evaluate a user's current competencies against targeted job requirements, precisely isolating the "gap."
3. **Learning Path Generator:** Aggregates and optimizes a curated sequence of educational courses, utilizing data from institutional partners and open-source learning platforms.
4. **Progress & Assessment Tracker:** Implements data analytics and gamification principles to ensure high retention rates and measurable competency validation.
5. **Job Guarantee System:** A functional module matching verified, "job-ready" candidates with pre-committed industrial partners.

### Technical Stack (Prototype)

- **Frontend Development:** React 18, TypeScript, Vite
- **Styling:** CSS3 (Modern, Responsive Design)

## 🚀 Getting Started

The current repository contains the initial structure and front-end prototype of the web application.

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [pnpm](https://pnpm.io/) (Fast, disk space efficient package manager)

### Installation & Execution

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Eakkachad/O-Guard.git
   cd O-Guard
   ```

2. **Navigate to the frontend application:**

   ```bash
   cd app
   ```

3. **Install dependencies:**

   ```bash
   pnpm install
   ```

4. **Run the development server:**
   ```bash
   pnpm run dev
   ```
   The application will be accessible at `http://localhost:5173` (or the port specified in your terminal).
